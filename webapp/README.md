Frontend Web Application for Commute page
---
[![Build Status](https://travis-ci.org/caroleolivier/commute-status.svg?branch=master)](https://travis-ci.org/caroleolivier/commute-status)

Set up
---
```
npm install
```

Build
---
```
npm run build
```

Test
---
```
npm run test
```

Start
---
```
npm run serve
```
Go to http://localhost:9000/
